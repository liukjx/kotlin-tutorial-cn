## 学习路径

### 初学者（预计 2-3 小时）
1. 基本语法 → 基本类型 → 控制流
2. 函数定义 → Lambda 表达式

### 进阶（预计 2-3 小时）
1. 类与继承 → 属性 → 接口
2. 数据类 → 密封类 → 泛型

### 实战（预计 1-2 小时）
1. 集合概述 → 过滤映射 → 排序分组
